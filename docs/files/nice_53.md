# NICE 53期 | 基于事实依据的自动化营销文案生成框架

## 主题

基于事实依据的自动化营销文案生成框架

## 时间

2025.3.30 10:30 am 

## 内容

论文:Grounded Persuasive Language Generation for Automated Marketing
链接：https://arxiv.org/abs/2502.16810

摘要
我们开发了一套基于大语言模型（LLM）的智能体框架，用于自动化生成具有说服力且基于事实的营销内容，并以房地产房源描述作为核心应用场景。该方法旨在使生成内容既贴合用户偏好，又能有效突出实用事实特征。该框架包含三个核心模块：（1）事实依据模块——模拟专家行为预测市场关注点；（2）个性化模块——根据用户偏好定制内容；（3）营销优化模块——确保事实准确性并融入地域特征。通过在房地产营销领域开展系统性用户实验，邀请潜在购房者作为目标用户进行评估。实验结果表明相较于人类专家撰写的描述，目标用户更喜爱我们方法生成的营销文案，且优势明显。研究结果表明，该基于 LLM 的智能体框架能够实现大规模精准的自动化营销，同时确保生成内容真实可靠。

报告议程：
1. 快速回顾：传统基于主观意见的说服方式
2. 基于事实依据的说服策略
3. 智能体工作流与用户界面设计
4. 效果评估
