# NICE 89期 | Agentic Web: Weaving the Next Web with AI Agents

## 主题

Agentic Web: Weaving the Next Web with AI Agents

## 时间

2025.9.8 10:00

## 内容

AI智能智能体的出现标志着互联网进入了一个新的阶段——“智能体网络”（Agentic Web）。在这个阶段，智能体之间可以自主地相互协作，规划并执行复杂任务，代替用户处理日常的数字操作，从而让用户得到解放，实现更加智能和自动化的网络体验。本文提出了一个结构化框架，帮助理解和构建智能体网络，回顾了这一转变从PC和移动互联网时代的演变，并探讨了支撑这一转型的关键技术。框架的核心是智能、互动和经济三个维度，这三者共同赋能了AI智能体在信息检索、推荐、规划和协作等方面的能力。文章还分析了在构建可扩展的智能系统时面临的架构和基础设施挑战，包括通信协议、协调策略以及新兴的“智能注意力经济”等概念。最后，本文讨论了智能系统带来的潜在应用、社会风险和治理问题，并提出了未来的研究方向，旨在发展一个由人类意图和自主智能体行为共同塑造的开放、安全和智能的生态系统。
