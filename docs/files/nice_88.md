# NICE 88期 | 一篇顶会论文的诞生：从零到一的实战经验分享

## 主题

一篇顶会论文的诞生：从零到一的实战经验分享

## 时间

2025.9.7 10:00

## 内容

很多做科研的同学，都会有类似的困惑：
资料和信息要去哪里找？
研究点子该怎么想出来？
论文写作为什么总是卡住？
图和表怎样画才直观、好看？
成果出来了，该怎么宣传？
中稿以后，又要怎样准备报告和展示？
其实，这些问题几乎每个人都遇到过。科研不仅仅是写论文，它是一条完整的流程：从最开始的一个小小“问号”，到最后成果发表、站上讲台的“叹号”。我们带大家走一遍“论文的成长之路”。从科研资讯的获取，到如何捕捉并筛选 idea；从论文结构和写作技巧，到图表设计与可视化表达；从成果的推广与宣传，到最后在会议上自信地展示。希望通过这一条龙的分享，让大家对科研的每一步都有更清楚的认识。
